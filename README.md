# PDF Sign Tool

A free, easy-to-use PDF signing tool built with ATOM Framework. Sign PDF documents instantly without any signup or hassle.

## Features

- ✅ **100% Free** - No hidden fees, no subscriptions
- ✅ **No Signup Required** - Start signing immediately
- ✅ **Secure & Private** - All processing happens in your browser
- ✅ **Easy to Use** - Simple drag-and-drop interface
- ✅ **Two Signature Options**:
  - Draw your signature with mouse or touch
  - Type your signature in cursive style
- ✅ **Instant Download** - Get your signed PDF immediately

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## How to Use

1. **Upload PDF**: Click the upload area or drag and drop your PDF file
2. **Choose Signature Type**: Select "Draw Signature" or "Type Signature"
3. **Add Signature**: 
   - Draw: Use your mouse or touch to draw your signature
   - Type: Enter your signature text
4. **Sign PDF**: Click the "Sign PDF" button
5. **Download**: Download your signed PDF file
